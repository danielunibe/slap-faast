# 🚀 FAAST (Slap!Faast)

**Número de Proyecto:** 60
**Estado:** En Desarrollo / Prototipo Hardware-Software
**Tipo:** Interacción Háptica / Control por Movimiento

## 📋 Resumen del Proyecto

**FAAST (Flexible Action and Articulated Skeleton Toolkit)**, o conocido dentro del entorno como "Slap!Faast", es un ambicioso proyecto centrado en la integración de hardware de mapeo corporal (específicamente Microsoft Kinect V1) con el control de PC e interfaces interactivas. La base de código y los recursos abarcan desde binarios interactivos hasta scripts de Python, PowerShell web, y recursos 3D (mesh).

## 🎯 Objetivos Principales
1. **Control de Interfaz mediante Movimiento:** Hacer uso del Kinect V1 para mapear gestos del usuario ("Slap") a inputs del sistema o acciones dentro de un entorno virtual.
2. **Experimentación Multiplataforma:** Integración de múltiples lenguajes y formatos (HTML/JS/CSS, Python, PowerShell) para coordinar el hardware con el software.

## 🛠 Entregables y Archivos Clave
- **Control de PC con Kinect V1:** Directorios y documentación (archivos en `documentos para codex`) para vincular los esqueletos detectados con acciones OS.
- **Slap!Faast:** Prototipo central que incluye lógica de interacción, assets visuales y de audio.
- **Empaquetado de Instalación:** Archivos `.zip` (`Cómo Crear Slap!Faast...zip`) y `.ps1.zip` que facilitan la replicabilidad del entorno e instalación.

## 🚀 Próximos Pasos
- Estabilizar los drivers y scripts de conexión del Kinect para asegurar consistencia en la respuesta física.
- Depurar las interfaces de usuario (HTML/JS) para un "dashboard" más moderno de calibración.
